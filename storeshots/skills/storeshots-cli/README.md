# storeshots-cli skill

Cursor Agent Skill for the [storeshots](../../README.md) marketing asset CLI.

## Install for Cursor

Copy or symlink into your personal skills directory:

```bash
mkdir -p ~/.cursor/skills
cp -R skills/storeshots-cli ~/.cursor/skills/storeshots-cli
```

Or from the utilities repo root:

```bash
ln -sf "$(pwd)/storeshots/skills/storeshots-cli" ~/.cursor/skills/storeshots-cli
```

Restart Cursor or start a new agent session so the skill is discovered.

## Triggers

The agent should load this skill when you mention storeshots, App Store screenshots, marketing assets from raw captures, BRAND.md extraction, print collateral, or paid ad layouts.
